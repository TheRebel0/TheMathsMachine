Introduction:

The Maths Machine - aka TMM is a simple analogue breadboard computer made by TheRebel0. This computer allows you to simulate physics using 2x potentiometers, allowing for the adjustment of position and velocity. This is an open soure device meaning that you can build it yourself via the schematic diagram or this tutorial.

Parts List:

The required parts to build it are shown below.

2x Potentiometer.

1x 10uf capacitor (or any reasonable size, this will change the lag of the integrator)

2x 10k resistor (again, you can pick any size)

1x Arduino (any model, this does not impact performance)

1x Breadboard

About 15 jumper wires (more than needed, this is for spares)

Things to consider:

Be careful to add a 10k resistor between your Arduino 5V and potentiometer to avoid shorting it.

Make sure that the capacitor has a path to GND so that you can turn the potentiometer down.

Building:

Here is the wiring.

5V -> + rail on breadboard

GND -> - rail on breadboard

POT1 left leg -> 10k -> + rail

POT1 right leg -> - rail

POT1 middle leg -> + Capacitor

POT2 left leg -> 10k -> + rail

POT2 right leg -> - rail

POT2 middle leg -> Analogue pin 1

Cap + leg -> A0

Cap - leg -> - rail

Cap + leg -> 10k -> - rail
